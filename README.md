# online-market-mini-game
상품들을 옵션에 따라 필터링하는 간단한 동적 페이지를 구현한 프로젝트입니다.

HTML, CSS, JS 기본 연습용 웹사이트 구현

## 구현 내용
1. HTML, CSS 파일 작성
2. 상품 정보를 저장하는 JSON file
3. JavaScript를 이용한 동적 요소 추가
4. 상단 아이콘 + 버튼 클릭시 발생할 이벤트 구현

## 기술 스택
HTML, CSS, JavaScript
